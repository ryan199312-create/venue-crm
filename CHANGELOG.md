# 更新日誌 (Changelog)

## [1.2.0] - 2024-04-19

### 新增 (Added)
- **指南與更新 (Docs & Updates)**: 新增專屬介面，用於展示系統更新歷史與使用指南。
- **更新提醒系統**: 在側邊欄加入動態提醒（小紅點），當系統有新版本發布時自動提示管理員。

### 優化 (Improved)
- **側邊欄導航**: 整合「指南與更新」標籤，並優化整體導航體驗。
- **版本管理**: 建立基於 `localStorage` 的版本追蹤機制。


## [1.1.0] - 2024-04-18

### 新增 (Added)
- **全新分頁式編輯介面**: 將 `EventFormModal` 拆分為多個獨立標籤頁，提升操作體驗與代碼維護性：
    - **基本資料 (Basic Details)**: 包含活動狀態、類型、日期、時間及客戶資訊。
    - **餐飲細節 (F&B)**: 菜單選擇、額外需求及加購項目。
    - **收費明細 (Billing)**: 自動計算總額、訂金及餘額，支持附加費用。
    - **物流安排 (Logistics)**: 新增活動流程 (Rundown)、旅遊巴安排、送貨安排與泊車資訊。
    - **場地佈置 (Venue)**: 支援場地平面圖檢視與照片管理。
    - **打印配置 (Print Config)**: 自定義各類文件顯示項目。
    - **內部備註 (Remarks)**: 提供獨立的內部備註空間，不會顯示於客戶文件中。
- **PDF 生成引擎**: 引入 `usePdfGenerator` Hook，整合 `jsPDF` 與 `html2canvas`，支援生成合約、報價單、發票、收據及內部單據。
- **文件管理**: `DocumentManager` 組件現在支援生成與下載多種文件格式。
- **自動化計算**: 整合 `vmsUtils.js` 實現最低消費 (Min Spend) 自動匹配與收費計算。

### 優化 (Improved)
- **UI 組件庫**: 統一 `src/components/ui.jsx` 中的輸入框、選擇框等基礎組件樣式。
- **表單驗證**: 強化了關鍵欄位的必填校驗。
- **動畫效果**: 使用 `framer-motion` 與 Tailwind CSS 增加介面過渡動畫。
- **移動端適配**: 優化 `AdminMobileHeader` 與側邊欄導航。

### 修復 (Fixed)
- 修正了 `DocumentRenderer` 在渲染中文內容時的字體與排版問題。
- 修復了場地平面圖 (`FloorplanViewer`) 的縮放與顯示錯誤。

### 核心更改 (Core Changes)
- **架構重構**: 核心組件模組化，降低 `AdminApp` 的複雜度。
- **Firebase Functions**: 更新後端函數以支援新的數據結構。
- **依賴更新**: 更新了 `framer-motion` 等核心套件。
